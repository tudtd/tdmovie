# TdMovie

TdMovie is a simple movie web build with React.js.

## Live Demo

Website: https://tdmovie.vercel.app/

## Technologies Used

React JS, SASS

## Api

LokLok.

## Features

- Watch Full HD movies with subtitles in many languages
- Search, suggest movies
- Watch list
